# AgentFlow Framework

This is the `.claude/` directory containing the AgentFlow framework.

**For full documentation, see: [docs/README.md](./docs/README.md)**

## Directory Structure

| Directory | Purpose |
|-----------|---------|
| [docs/](./docs/) | Framework documentation (entry point) |
| [agents/](./agents/) | Specialized task execution agents |
| [skills/](./skills/) | Process and expertise knowledge (includes orchestration) |
| [commands/](./commands/) | Slash commands |
| [hooks/](./hooks/) | Claude Code hooks |
| [scripts/](./scripts/) | Validation and setup scripts |
| [templates/](./templates/) | File templates |
| [lib/](./lib/) | Shared TypeScript utilities |
| [work/](./work/) | Local working files (not synced) |

## Key Files

| File | Purpose |
|------|---------|
| [CLAUDE-agentflow.md](./CLAUDE-agentflow.md) | Framework instructions for Claude |
| [settings.json](./settings.json) | Hook and agent configuration |
