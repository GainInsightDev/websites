# Hello World Application Template

This template provides a minimal Next.js + AWS Amplify Gen 2 application that can be used to verify infrastructure setup.

## What's Included

- **Next.js 15** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **Amplify Gen 2 Backend** - GraphQL API with Lambda function
- **Pre-configured build** - `amplify.yml` ready for deployment

## Features

The Hello World page demonstrates:
- GraphQL API integration
- Loading state handling
- Error handling
- Timestamp display
- Guest (unauthenticated) access

## Usage

### 1. Copy to your project

```bash
cp -r .claude/templates/gaininsight-standard/hello-world-app/* /path/to/your/project/
```

### 2. Replace placeholders

Replace `{{project-name}}` with your actual project name in:
- `package.json` (name field)
- `src/app/layout.tsx` (title metadata)

### 3. Install dependencies

```bash
npm install
```

### 4. Run locally with sandbox

```bash
# Terminal 1: Start Amplify sandbox
doppler run --project YOUR_PROJECT --config dev -- npx ampx sandbox

# Terminal 2: Start Next.js dev server
npm run dev
```

### 5. Verify it works

Visit http://localhost:3000 and you should see:
- "Hello World" heading
- API message after loading
- Timestamp from the backend
