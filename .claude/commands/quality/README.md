# Quality Commands

Commands for maintaining documentation and code quality in AgentFlow projects.

## Available Commands

### [/quality:curator](./curator.md)
Enforces DRY across documentation by finding duplicates, identifying authoritative sources, and replacing copies with references. Weekly/sprint maintenance.

### [/quality:docs](./docs.md)
Validates documentation standards, repairs frontmatter, and ensures documentation completeness. Run before commits.
